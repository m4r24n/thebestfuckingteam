# TBFT v0.4 — Mellow Material Theme

## Interface sizing

The Small, Medium, and Large options now scale the entire interface rather than only the navigation. The selected size changes the root visual scale, including:

- all typography
- task cards and task columns
- buttons and completion controls
- spacing and gaps
- sidebar and top bar
- calendar, project, activity, and settings screens
- dialogs, form fields, and thread messages
- borders, radii, and shadows

The layout keeps the same proportions at every size.

## Theme redesign

- Replaced the black navigation with a soft theme-aware sidebar.
- Added six mellow palettes: Mustard, Muted green, Mellow blue, Clay, Soft mauve, and Blue green.
- The selected theme colors the navigation, active states, buttons, progress, and focus outlines.
- Changed the workspace from white to warm off-white.
- Added very subtle matte paper grain and painted-surface variation without external image assets.
- Added light inset highlights and restrained shadows to create a soft embossed material effect.
- Updated task cards, dialogs, settings, calendar cells, and project surfaces to use the same material system.

## Compatibility

Existing local tasks and projects are preserved. Older black and saturated accent selections are automatically migrated to their closest mellow palette.
