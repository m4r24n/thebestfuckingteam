# TBFT v0.3 — Clean Start & Appearance Controls

- Removed all prefilled demo tasks, projects, messages, and activity.
- New installations open on a clean two-column daily board.
- Added three interface sizes: Small, Medium, and Large.
- Added six restrained accent-color presets.
- Added database fields for interface size and accent color.
- Renamed demo reset controls to local workspace controls.
- Updated the browser-storage key so the deployed preview starts clean.

The application still uses browser-local storage. Supabase authentication, workspaces, and realtime synchronization are the next production phase.
